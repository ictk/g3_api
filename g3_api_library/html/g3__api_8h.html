OUT void *  data_structure,
OUT int  structure_size 
)

Read value from the setup area, key area and user data area.

Parameters
key_index: Key sector index ( p1 )
area_type: The type of area to read ( p2_lower ) It can be one of the following values :
  • SETUP_AREA
  • KEY_AREA
  • DATA_AREA_0
  • DATA_AREA_1
rw_type: Read operation mode(p2_upper) It can be one of the following values :
  • PLAIN_TEXT - plain read
  • CBC, CTR, SESSION_KEY_CBC, SESSION_KEY_CTR - encrypted read
  • CBC_with_MAC, CTR_with_MAC, SESSION_KEY_CBC_with_MAC, SESSION_KEY_CTR_with_MAC - encrypted read with MAC
  • MASKED - masked read
data: Input data only for masked operation(rw_type -> MASKED)
data_size: The size of the mask
data_structure: Read data. it can be one of the following structures :
  • ST_RW_DATA : Data[32] - plain read, masked read
  • ST_RW_DATA_WITH_IV : Data[32] || IV[16] - encrypted read
  • ST_RW_DATA_WITH_IV_MAC : Data[32] || IV[16] || MAC[16] - encrypted read with MAC
structure_size: The size of data_structure
Returns
