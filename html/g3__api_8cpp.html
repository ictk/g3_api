IN const void *  Q_b,
IN int  Q_b_struct_size,
IN const ST_ECDH_RANDOM *  st_ecdh_random,
OUT ST_ECC_PUBLIC *  Q_chip,
OUT void *  ecdh_value,
OUT int  ecdh_value_struct_size 
)

Used for session key agreement.

Parameters
en_ecdh_mode: The mode of ECDH instruction It can be one of the following values :
  • NORMAL_ECDH, GEN_TLS_BLOCK, SET_TLS_SESSION_KEY
Q_b: The type of public key It can be one of the following values :
  • ST_ECC_PUBLIC : puk[64]
  • ST_ECC_PUBLIC_COMPRESS : puk[33]
Q_b_struct_size: The size of public key
st_ecdh_random: ServerHello.random[32] || ClientHello.random[32]
Q_chip: Command outputs, public key chip(dchipG)[64]
ecdh_value: Command outputs. It depends on the ECDH mode and can be one of the following values :
  • NORMAL_ECDH : pre_master_secret[32]
  • GEN_TLS_BLOCK : TLS_key_block[128]
  • SET_TLS_SESSION_KEY : client_write_IV[16] || server_write_IV[16]
ecdh_value_struct_size: The size of ecdh_value
Returns
G3_API_RESULT
